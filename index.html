📌 Project Overview

This project demonstrates a complete CI/CD pipeline using Jenkins, where code pushed to GitHub is automatically built and deployed to a web server. The pipeline automates the entire software delivery process, reducing manual interventions and ensuring consistent deployments.

Key Features:

🔄 Automation

Code changes trigger automatic builds and deployments

⚡ Fast Deployment

Reduces deployment time from manual hours to minutes

📊 Reliability

Consistent and reproducible deployment process

🔐 Security

Automated testing and credential management

🏗️ Architecture Workflow

The pipeline follows a sequential flow from development to production:

Developer (VS Code) ↓ GitHub Repository ↓ (Webhook Trigger) Jenkins Server (Public IP) ↓ Build & Deployment ↓ Web Server (Live Website)

Workflow Steps:

  1. Developer writes code in VS Code
  2. Code is pushed to GitHub repository
  3. GitHub webhook automatically triggers Jenkins
  4. Jenkins pulls the latest code from repository
  5. Jenkins executes build and deployment process
  6. Website is updated automatically on web server

🔧 Setup & Configuration Steps

1

Jenkins Setup

Installed Jenkins on Linux server and configured it to be accessible via a public IP address.

  • Installed Jenkins on Linux server
  • Configured Jenkins and exposed via Public IP
  • Installed required plugins: Git Plugin, Pipeline Plugin
2

GitHub Integration

Created repository and configured authentication for Jenkins to access GitHub.

git init git add . git commit -m "Initial commit" git push origin main
3

Webhook Configuration

Configured GitHub webhook to automatically trigger Jenkins on every code push.

http://<JENKINS_PUBLIC_IP>:8080/github-webhook/

✔ Automatically triggers Jenkins on every push

4

Jenkins Job Configuration

Created a Jenkins job and configured build triggers and GitHub credentials.

  • Created Freestyle/Pipeline job
  • Configured GitHub repo URL
  • Set branch to: main
  • Added credentials (PAT)
  • Set build trigger: GitHub webhook
5

Build & Deployment

Jenkins executes deployment commands that update the website automatically.

rm -f /var/www/html/index.html rm -f /var/www/html/index.nginx-debian.html cp index.html /var/www/html/

✔ Automatically updates website
